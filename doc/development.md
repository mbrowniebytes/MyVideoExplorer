[set autocrlf to input](https://stackoverflow.com/a/4425433)

~[user dir]/.gitconfig

```
[user]
	name = [your github name]
	email = [your github name]@users.noreply.github.com

	autocrlf = input
```

### install python 3.14+

install [python for windows](https://www.python.org/downloads/windows/)

install [git for windows](https://git-scm.com/install/windows)  
recommended use git bash in windows terminal

recommended ide [PyCharm](https://www.jetbrains.com/pycharm/)

### clone repo

> git clone [repo]

### initialize a python env

using [venv](https://pythonguides.com/virtual-environments-python/)
> py -3.14 -m venv .venv

### activate env

> source .venv/Scripts/activate

### install packages

using [pip](https://pip.pypa.io/en/stable/getting-started/)
> pip install -e .  
> pip install -e .[dev]

### list outdated packages

> pip list --outdated

### upgrade pip

> py -m pip install --upgrade pip

### update lock file pylock.toml

> py -m pip lock -e .

### install from lock file pylock.toml

skip for now, errors "cannot be installed when requiring hashes"
> pip install -r pylock.toml
